# Todo List

A simple todo list application built using HTML, CSS, and JavaScript.

[Project Link]
(https://ammar-amin-todolist.netlify.app/)

## Features

- Add new tasks to the list.
- Mark tasks as completed.
- Delete tasks from the list.

- JavaScript
